908. Smallest Range I

这道题一开始没看懂，其实不难

因为范围是 [-K, K]，所以，找到 A 中的 最大值 maxB 和 最小值 minB，然后我们计算差值就可以了，差值就只 maxB - K - (minB + K)，也就是 res = maxB - minB - 2 * K，如果 res > 0 返回即可，小于 0 的话，返回 0 即可。
